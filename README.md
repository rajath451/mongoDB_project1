 Registration Form Project

A simple registration form built using **Node.js**, **Express**, and **MongoDB**.  
This project demonstrates how to connect a frontend form to a backend server and store user data in a database.

---

Features
- User registration form (Name, Email, Password)
- Backend with Express.js
- MongoDB integration for storing user data
- Basic validation and error handling

Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
